Task Monitor

A task monitor can remind users about various appointments or events happening each day or week. This type of software program or app can function like a digital planner, an alarm clock or both. Task monitors typically use reminder software powered by artificially intelligent chatbots that can send notifications.

Portfolio website

Software developers who are also interested in web development might consider building their own portfolio website. Most portfolio websites use a combination of HTML, CSS and JavaScript coding languages. A portfolio website is a great practical project, as many professionals in the tech industry often use a digital portfolio when applying for various opportunities.

Bandwidth monitor

A bandwidth monitor can keep track of how much data a user downloads from or uploads to the internet. Consider developing a bandwidth monitor that sends regular notifications or emails about a user's weekly or monthly data usage. You can create a bandwidth monitor as a live CLI tool.
